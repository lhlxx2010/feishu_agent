### 全局影响评估

**目标：** 利用 Gemini 的超大上下文，像扫描仪一样检查这次修改会不会“牵一发而动全身”。

**操作提示：**
在使用这个 Prompt 前，请务必先将你的**整个项目代码**（或相关文件夹）打包成 Zip 上传，或者把目录结构和相关文件内容一次性发给 Gemini。

**提示词 (Copy 进 Gemini):**

```markdown
# Role
你就是这个项目的 Technical Lead，你熟悉这整个代码库的每一个角落。

# Context
我已经上传了当前项目的完整代码库。我刚刚对文件 `[修改的文件路径，例如: core/api_client.py]` 进行了修改。
